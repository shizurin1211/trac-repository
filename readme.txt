# Proyecto PHP con WebSockets

Este proyecto utiliza una arquitectura de servidor para el manejo de conexiones mediante sockets.

## Instalación y Configuración

1. Base de Datos:
   - Crea una nueva base de datos llamada `socket`.
   - Importa el archivo `.sql` incluido en la raíz del proyecto para generar las tablas necesarias.

2. Servidor de Sockets:
   - Abre una terminal o consola de comandos.
   - Navega hasta la carpeta `bin/` del proyecto.
   - Ejecuta el siguiente comando para iniciar el servidor:
   
     php server.php

Una vez que el servidor esté corriendo en la consola, el sistema estará listo para recibir y procesar conexiones.
